# StudyFlow

StudyFlow is a student-focused planning app designed to help manage academic workload, reduce stress, and prioritize tasks.

## Features
- Task tracking (To Do / In Progress / Done)
- Smart task recommendation based on workload
- Stress score system
- Weekly calendar view
- Built-in focus reset tool

 The app opens with sample academic tasks so users can immediately test the smart suggestion, workload scoring, and calendar features.

## Tech
- HTML
- CSS
- JavaScript

## Why I built this
I wanted to create a planner that doesn’t just track tasks, but actually helps students decide what to start next based on difficulty and workload.
